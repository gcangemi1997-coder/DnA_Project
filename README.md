
# 🌐 DnA – Accessible Scientific Outreach

> “La tua **Conoscenza**, la nostra **Missione**.”

DnA is a static landing page designed to showcase a project focused on clear, accessible, and trustworthy scientific communication.  
The page targets a general audience and aims to make scientific content easier to understand while fighting misinformation.

---

## 📍 Live Demo 👉 [https://gcangemi1997-coder.github.io/DnA_Project/)

---

## ✨ Features

- 🎯 Single-page layout with a clean landing page structure.
- 🧬 Branded header with logo and the main tagline:
  - <em>“La tua Conoscenza, la nostra Missione!”</em>
- 🧭 Internal navigation bar with anchor links:
  - Chi Siamo, Contatti, FAQ.
- 📱 Social section with links to:
  - Facebook, LinkedIn, Instagram.
- 📚 “Mission” section focused on:
  - Accessibility of scientific knowledge.
  - Clarity and reliability of information.
- ⚖️ Footer with years range <code>2024–2026</code> and legal links:
  - Terms, Privacy Policy, Cookies.
