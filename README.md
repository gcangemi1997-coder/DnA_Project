# DnA – Accessible Scientific Outreach

DnA is a simple static landing page designed to present a project focused on clear and accessible scientific communication.  
The goal is to make scientific information more understandable and reliable for a general audience, avoiding misinformation and incomplete content.

## Demo ----> (https://gcangemi1997-coder.github.io/DnA_Project/)

## Features

- Single-page layout with a landing-page structure.
- Header with logo, main title “DnA” and tagline “La tua Conoscenza, la nostra Missione!”.
- Internal navigation bar with anchor links: Chi Siamo, Contatti, FAQ.
- Social section with links to Facebook, LinkedIn and Instagram.
- “Mission” section describing the purpose of making scientific knowledge accessible and fighting misinformation.
- Footer with reference years (2024–2026) and links to Terms, Privacy Policy and Cookies.

## Tech stack

- **HTML5** for the page structure.
- **CSS3** inline in the same HTML file for styling (layout, colors, typography, etc.).

No external libraries or frameworks are required.
